原理、技术与应用

土地科学与技术学院 土壤与水系

为什么研究土壤水热盐？

  • 核心地位: 土壤是地球关键带的核心，水热盐是重要的物理过程。
  • 对植物生长的影响:
    • 水: 光合作用原料、营养物质溶剂和载体。
    • 热: 影响种子萌发、根系生长和微生物活动。
    • 盐: 影响渗透势，高盐胁迫抑制生长。
  • 对生态系统的影响:
    • 水平衡: 决定降水在“产流-入渗-蒸发”中的分配。
    • 能量平衡: 土壤热通量是地表能量平衡的关键分量。
    • 物质循环: 影响碳氮循环、温室气体排放。
  • 在生产实践中的应用: 精准灌溉、盐碱地改良、旱情监测等。

核心概念 - 土壤水

土壤含水量 (Soil Water Content)

  • 质量含水量 (θg): 水质量 / 干土质量。 (单位: g/g)
  • 体积含水量 (θv): 水体积 / 土壤总体积。 (单位: cm³/cm³)
  • 换算: θv = θg × (ρb / ρw)，其中 ρb 为土壤容重，ρw 为水的密度。
  • 土壤容重 (ρb): ρb = 干土质量 / 土壤总体积。 (单位: g/cm³)
  • 土壤孔隙度 (f): f = 1 - (ρb / ρs)，其中 ρs 为土壤颗粒密度。

土壤水势 (Soil Water Potential, ψ)

  • 衡量土壤水能量状态的指标，决定水流方向。(单位: Pa, bar, cm水头)
  • 组成: 基质势 (ψm) + 渗透势 (ψo) + 重力势 (ψg) + 压力势 (ψp)。
  • 田间土壤水主要受 基质势 和 重力势 控制。

核心概念 - 土壤热

土壤温度 (Soil Temperature)

  • 土壤热状况最直观的指标，影响土壤物理、化学和生物过程的速率。
  • 呈周期性变化（日/年），随深度增加，振幅减小，相位延迟。
不同深度土壤温度周期变化公式：
地表温度（z=0）：
$$T(0, t) = T_0 + A_0 \sin(\omega t)$$
z深度温度：
$$T(z, t) = T_0 + A_0 e^{-z/D} \sin(\omega t - z/D)$$
其中：T(z, t)为z深度t时刻温度，T_0为年/日均温，A_0为地表温度振幅，D为阻尼深度，ω为角频率。

核心概念 - 土壤热

土壤热通量 (Soil Heat Flux, G)

  • 单位时间、单位面积上通过土壤的热量。
  • 地表能量平衡方程关键分量: Rn = G + H + LE
  • 其中：Rn为净辐射，G为土壤热通量，H为显热通量，LE为潜热通量。

土壤热特性

  • 热容量: 单位体积土壤温度升高1°C所需吸收的热量。 (单位: J/(m³·°C))
  • 热导率: 单位时间内，在单位温度梯度下，单位截面积土壤能传递的热量。 (单位: W/(m·°C))
  • 热扩散率: 土壤温度变化扩散速度的参数，等于热导率与热容量的比值。 (单位: m²/s)
  • 关键影响因素: 土壤含水量、容重、有机质、质地。

核心概念 - 土壤盐

土壤盐分 (Soil Salinity)

  • 定义: 土壤中所有可溶性盐类的总称。
  • 衡量指标: 总含盐量 (%) 或 土壤溶液电导率 (EC)。
  • 电导率 (EC): 最常用，单位: dS/m, mS/cm。

重要EC指标区分

  • ECe: 饱和土浆浸提液电导率 (实验室标准方法)。
  • EC1:5: 1:5土水比浸提液电导率 (快速测定)。
  • ECw: 土壤孔隙水电导率 (作物根系真实环境)。
  • ECa (或 ECb): 土壤表观/体积电导率 (田间传感器直接测量)。

测量方法分类

按原理: 直接法 vs. 间接法
按地点: 实验室法 vs. 田间原位法 (In-situ)
按自动化程度: 人工测量 vs. 自动监测

本课程重点: 田间原位 (In-situ) 监测方法

第二部分：土壤水分测定方法

直接法 - 烘干法 (Gravimetric Method)

原理: 通过105°C烘箱烘干去除水分，计算失重量与干土重的比值。

设备: 土钻、环刀、铝盒、天平 (0.01g)、烘箱。

优点:
  • 标准方法 (Standard Method)，最准确。
  • 仪器简单，成本低。
  • 是校准其他间接方法的基准。
缺点:
  • 破坏性取样，无法定点重复测量。
  • 耗时耗力 (>12小时)。
  • 无法获得实时、连续数据。
  • 空间代表性有限。

间接法 - 宇宙射线中子传感器 (Cosmic Ray Neutron Sensor)

原理与应用

原理: 放射源发射快中子，与水中的氢核碰撞后能量衰减为慢中子，探测器记录慢中子数，其与体积含水量正相关。

优点:
  • 测量范围大 (水平范围可达 200-300 m 半径，有效深度约 15-30 cm)。
  • 结果稳定，不受盐分和温度影响。
  • 可重复测量剖面含水量。
缺点:
  • 放射性源: 需严格安全许可和操作。
  • 价格昂贵，设备笨重。
  • 近地表 (0-20cm) 测量误差大。
  • 对有机质敏感。
  • 正逐渐被更安全的技术取代。

间接法 - 时域反射法 (Time Domain Reflectometry, TDR)

核心原理: 土壤的表观介电常数 (Ka) 主要由液态水 (ε≈80) 决定。通过测量电磁波在土壤探针中的传播时间，反算Ka，进而得到含水量θv。

测量过程:

  1. TDR仪器产生一个高频电磁脉冲。
  2. 脉冲沿同轴电缆传输到插入土壤的金属探针 (波导杆)。
  3. 脉冲在波导杆末端反射回仪器。
  4. 仪器精确测量脉冲的往返传播时间 (Δt)。
  5. 脉冲传播速度 (v) 取决于Ka: v = c / √Ka (c为光速)。
  6. 通过 Ka = (c✖️Δt / 2L)² 计算出Ka，进而换算为θv。

TDR的优缺点

优点:

  • 高精度、高准确性。
  • 响应速度快 (秒级)，可连续监测。
  • 对盐分不敏感 (测量的是时间)。
  • 校准稳定性好，通用性校准方程适用性广。
  • 可同时测量土壤体电导率 (ECa)。

缺点:

  • 价格昂贵。
  • 波形分析复杂 (现代仪器已内置算法)。
  • 在黏粒或有机质含量高的土壤中，需要特定校准。
  • 探针安装要求高，必须与土壤紧密接触。

间接法 - 频域法 (Frequency Domain Reflectometry, FDR) / 电容法

核心原理: 同样基于含水量与介电常数的关系。但其构成一个振荡电路，通过测量电路振荡频率随土壤介电常数的变化来反推含水量。

优点:

  • 成本远低于TDR。
  • 功耗低，技术简单，易于集成。
  • 探头形状多样，安装灵活。

缺点:

  • 对土壤盐分、温度和质地更敏感。
  • 测量体积相对TDR较小。
  • 通常需要针对特定土壤进行校准才能达到高精度。

第三部分：土壤热参数测定

土壤温度测量

电子传感器:

  • 热电偶 (Thermocouple): 价格低，精度一般。
  • 热敏电阻 (Thermistor): 灵敏度高、响应快，但非线性。
  • 电阻温度检测器 (Resistance Temperature Detector, RTD): 精度高、稳定性好，但价格较高。

红外成像法

  • 红外成像 (Infrared Thermography): 非接触式、快速覆盖，受环境影响、只能测量表面温度、成本高。
  • 分布式光纤温度传感 (Fiber Optic Distributed Temperature Sensing, DTS): 大范围和深度、精度高、耐腐蚀，安装复杂、成本高。

安装要点:

  • 方向: 水平插入，减少热传导误差。
  • 接触: 确保传感器与土壤良好接触。

土壤热通量测量 (Soil Heat Flux Plate)

原理:

  • 基于傅里叶热传导定律: G = -λ (dT/dz)。
  • 热通量板是一种已知热导率的扁平物体，通过测量其上下表面的微小温差 (ΔT)，计算出通过板的热通量Gplate。

结构:

  • 扁平的热电堆 (多个串联的热电偶) 封装在保护材料中。
  • 输出与温差成正比的毫伏电压信号。

热通量板的安装与数据校正

安装要点:

  • 通常埋设在地下2-10 cm深度。
  • 水平放置，与土壤紧密接触。
  • 多个板取平均值以克服空间异质性。

数据校正 (重要!):

  • 热通量板忽略了其上方土壤层的热量储存变化 (ΔS)。
  • 地表真实热通量 Gsurface = Gplate + ΔS
  • ΔS 需要通过测量板上方的土壤温度和含水量的变化来计算。

第四部分：土壤盐分测定

实验室标准法

饱和土浆浸提液法 (Saturated Paste Extract)

  • 步骤: 风干 → 制备饱和土浆 → 平衡静置 → 真空抽滤/离心 → 测浸提液电导率(ECe)。
  • 意义: 国际公认的土壤盐渍化分级标准，校准田间方法的基准。

土水比浸提液法 (1:1, 1:5)

  • 步骤简单快速，但结果受土壤质地影响大。

田间原位法 - 温纳法（四极等距法）

原理:

  • 外侧两电极施加交流电流(I)，内侧两电极测量电势差(ΔV)。
  • 根据欧姆定律和电极构型计算出土壤体电导率 ECa。

应用:

  • 可快速进行大面积盐分普查。
  • 测量深度与电极间距有关。

田间原位法 - 电磁感应法 (EMI)

代表仪器: Geonics EM38

原理:

发射线圈(Tx)产生一次磁场 → 在导电土壤中感生出与ECa成正比的涡电流 → 涡电流产生二次磁场 → 接收线圈(Rx)探测二次磁场，计算出ECa。

优点:

  • 非接触式测量，快速高效。
  • 可测量不同深度 (改变仪器方向)。
  • 非常适合大尺度土壤盐分制图。

缺点:

  • 测量精度受盐分、温度质地影响。
  • 需要针对不同土壤类型进行校准。
  • 价格较高，设备笨重。
  • 深度分辨率有限。

利用水分传感器同步测盐

原理:

TDR和FDR传感器在测量含水量的同时，也能测量土壤的电导特性，即土壤表观电导率 (ECa)。

换算模型:

ECa受含水量(θv)和孔隙水电导率(ECw)共同影响。需要利用特定模型 (如Hilhorst模型) 将ECa、θv和温度数据，转换为对农业更有意义的孔隙水电导率ECw。

Hilhorst模型表达式:

Hilhorst模型用于估算孔隙水电导率：

$$\sigma _{p}=\frac{\varepsilon _{p}\sigma _{b}}{\varepsilon _{b}-\varepsilon _{\sigma b=0}}$$

其中：σp：孔隙水电导率 (S/m); εp：水的介电常数; σb：土壤体积电导率 (S/m); εb：土壤体积介电常数; εσb=0：当电导率为0时的土壤体积介电常数

第五部分：综合监测与展望

集成传感器与监测网络

  • 多参数集成传感器: 在同一点同步获取水分、温度、电导率，研究耦合效应。
  • 自动监测站 (物联网, IoT): 实现长期、无人值守、高频率的连续监测。
  • 站点布设与网络设计:
    • 代表性: 覆盖主要土壤类型、地形和利用方式。
    • 空间变异性: 捕捉整体空间格局。
    • 分层监测: 监测剖面动态。

数据处理与质量控制

  • 数据采集与存储: 设置合理频率，建立数据库。
  • 质量控制 (QC):
    • 剔除异常值 (超出物理范围、传感器故障)。
    • 数据插补与平滑。
    • 定期进行田间原位校准（如用烘干法校准水分传感器）。
  • 数据分析与可视化: 时间序列分析、剖面动态图、空间插值制图等。

总结与未来趋势

总结:

  • 方法的选择取决于研究目的、精度、预算和时空尺度。
  • 理解物理原理是正确使用和解读数据的关键。
  • 交叉验证和校准是保证数据质量的核心环节。

未来趋势:

  • 低成本、智能传感器。
  • 物联网 (IoT) 与无线传感器网络 (WSN)。
  • 多源数据融合 (地面-无人机-卫星)。
  • 模型-数据融合 (监测+模拟)。

实验设备介绍

TEROS 12 多参数土壤传感器

技术参数:

  • 体积含水量: 精度 ±0.03 m³/m³，分辨率 0.001 m³/m³
  • 温度: 精度 ±0.5°C，分辨率 0.1°C，范围 -40~60°C
  • 电导率: 精度 ±5%，范围 0~20 dS/m
  • 供电: 3-15V DC，功耗 <3mA

技术优势:

  • 基于70MHz频域反射技术，测量稳定可靠
  • 三参数一体化设计，安装简便
  • 抗盐碱、抗腐蚀，适合长期野外监测
  • 响应速度快，可实现高频连续监测
